TIKET

A small ticking and time tracking system for a college computer repair shop

View on GitHub Download .zip Download .tar.gz

Welcome to TIKET.

TIKET is a small project using Laravel as the framework, it combines features of a public tickting system and real time time management applications to create the perfect workshop scheduler. This web application was create for Basingstoke College of Technology, by Benjamin Lack to use as OOP project for assessment.

Based on Laravel

With the power of the laravel framework behind this project, it will continued to be updated periodically. This also provides the perfect ground for cloning, as the project can be easily re-shapped into a new environment.

Author and Contributor

Thanks to @Xalaxis for giving ideas and the scope for the project. He has been a huge help with this system. Benjamin Lack is the sole designer and creator of this web application, he has made it to provide a complete system for his college's tech repair shop.
